# Connecting-to-Cloudflare-WARP
Settings for a succeful connection to Cloudflare WARP.
I recommend that you reboot your device only after completing all the steps.  Next, run 'general.bat' on behalf of the administrator, if WARP does not connect, then run 'general(ALT).bat' and so on.
Link to download the working version of WARP: https://drive.google.com/file/d/1ej9fsse4zcOmJ6Gu-fYjyU7_q6IyNGxR/view?usp=sharing
